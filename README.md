# R Analytics Portfolio

This repository contains practical R projects focused on insurance analytics, reporting automation, Shiny dashboards, Excel-connected workflows, and structured data preparation.

The projects were built as preparation for an R-heavy analyst/coder working student role in the insurance and reinsurance industry.

## Goal

The goal of this repository is to demonstrate practical, job-relevant R skills, especially in:

- R programming
- tidyverse data wrangling
- reporting and visualization
- ggplot2
- Shiny dashboards
- Excel import/export workflows
- Git-based project organization
- SQL/Postgres fundamentals
- PDF/OCR/API-style workflows
- practical insurance analytics

## Role Focus

Based on information from a current working student in a similar role, the work appears to be focused more on practical analytics and tool-building than on deep theoretical actuarial mathematics.

The most relevant work areas are:

- data cleaning
- reporting automation
- dashboard development
- Excel-connected workflows
- R-based tool building
- structured reporting
- practical insurance KPI calculation

## Main Preparation Targets

By the end of this preparation project, I want to be able to:

## Portfolio Projects

### 1. Insurance Reporting Dashboard in R/Shiny

A practical Shiny dashboard project using claims and premium-style data.

The dashboard includes:

- data cleaning
- quarterly summaries
- insurance KPI calculations
- claim count analysis
- severity analysis
- premium and loss ratio analysis
- filters and interactive dashboard outputs
- exportable reporting outputs

Project folder:

```text
09_portfolio_projects/project_01_insurance_dashboard/
```

### 2. PDF/OCR/API to Structured Reporting Workflow

A smaller supporting project that simulates turning semi-structured document-style information into clean reporting tables.

The workflow includes:

- document-style input
- text extraction/parsing logic
- claim table creation
- summary report generation
- Excel and CSV export

Project folder:

```text
09_portfolio_projects/project_02_pdf_ocr_pipeline/
```
